# Tops
实时热榜消息助手，可快速获取各大平台话题榜单。

## 介绍
本插件主要整合微博热搜、GitHub实时榜单等数据，以列表形式展示，方便我们**快速获取最新的消息 or 项目**，使用**鼠标右键-固定**将本插件固定至标签栏食用效果更佳，后续也将不断地进行维护更新。

## 主要功能
- 实时微博热搜
- 实时GitHub Trending榜单

## 安装方法
可以下载CRX文件手动安装（不建议）：[下载地址](https://github.com/zchengb/tops/releases)
